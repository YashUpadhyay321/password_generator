# 🔐 Password Generator

A **secure and customizable Password Generator** built using **HTML, CSS, Bootstrap, and JavaScript**. This tool generates strong passwords based on user preferences.

## 🚀 Features

- **Generate Strong Passwords** – Randomly creates secure passwords  
- **Customizable Length** – Choose the desired password length  
- **Include/Exclude Characters** – Options for uppercase, lowercase, numbers, and symbols  
- **Copy to Clipboard** – Easily copy the generated password  
- **Responsive Design** – Works on desktops, tablets, and mobiles  

## 🛠️ Technologies Used

- **HTML** – Structure of the webpage  
- **CSS** – Styling for a clean UI  
- **Bootstrap** – Ensures a responsive layout  
- **JavaScript** – Generates passwords dynamically  

🎯 Usage
    Select Password Length – Use the slider or input field to set the length.
    Choose Character Options – Enable/disable uppercase, lowercase, numbers, and symbols.
    Click "Generate Password" – A new secure password will be created.
    Copy to Clipboard – Click the copy button to use the password anywhere.
